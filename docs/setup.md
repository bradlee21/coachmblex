# Manual GitHub Setup (Phase 0)

1. Create a new GitHub repository named `coach-mblex` (no README, no .gitignore, no license).
2. From this local folder, set the remote:
   ```bash
   git remote add origin https://github.com/<YOUR_GITHUB_USERNAME>/coach-mblex.git
   ```
3. Push the `main` branch:
   ```bash
   git push -u origin main
   ```
4. Confirm files appear in GitHub, including `docs/PROJECT.md`, `docs/setup.md`, and `docs/phase-0-vercel.md`.
